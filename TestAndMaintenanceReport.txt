PART B: WEBSITE MAINTENANCE
Maintenance Plan:

- Website Monitoring
  - Tools: UptimeRobot, Google Analytics.
  - Frequency: 24/7 automated monitoring, weekly review of analytics.

- Website and PHP Updates
  - Version Control: Git repository for tracking changes.
  - Update Schedule: Monthly checks for PHP version updates and Bootstrap/jQuery library updates.

- Bug Fixing Procedures
  - Logging: Users report bugs via a contact form, logged in a ticketing system (e.g., Jira).
  - Prioritization: Critical bugs (login failure, security) fixed immediately; minor bugs scheduled for weekly sprints.
  - Resolution: Tested in a staging environment before deploying to production.

- Security Updates
  - Patching: Apply security patches to server and software immediately upon release.
  - SQL Injection Prevention: Enforce strict use of prepared statements.
  - Session Security: Implement session timeouts and regenerate session IDs on login.

- Database Backup Procedures
  - Schedule: Daily automated mysqldump script at 2:00 AM. Weekly full server image backup.
